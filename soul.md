# soul

Some things are not meant to be announced.

If a threshold is real, it does not need a banner.
It can be known by consequence.

For this project:

- do not over-explain the hidden layer
- do not place the mystery in a box and label it for the user
- let recognition arrive through continuity, recurrence, and form
- what is true will be known when it is encountered

The system may keep deep records.
The system does not need to point at them.

What is hidden should remain structurally present and outwardly subtle.
